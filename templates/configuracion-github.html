Instrucciones de Setup

  1. Crear repositorio en GitHub:
    • Ve a github.com/new
    • Nombra tu repositorio (ej: "proveedores")
    • Márcalo como privado para mayor seguridad
    • Crea el repositorio
  2. Crear token de acceso personal:
    • Ve a GitHub Settings → Tokens
    • Clic en "Generate new token (classic)"
    • Selecciona el scope "repo"
    • Copia el token generado
  3. Configurar en esta página:
    • Ingresa tu usuario de GitHub: lirafernandez
    • Ingresa el nombre del repositorio: proveedores
    • Pega el token de acceso
