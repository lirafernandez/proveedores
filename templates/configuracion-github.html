Advertencia de Seguridad

Su token de acceso se guardará en el almacenamiento local de su navegador. Esto es conveniente, pero puede ser un riesgo de seguridad.

Para mayor seguridad, use un token con permisos limitados (solo `repo`) y considere revocarlo cuando ya no lo necesite.

Instrucciones

  1. Crear Repositorio: Crea un repositorio privado en GitHub para almacenar tus datos de forma segura.
  2. Generar Token: Crea un nuevo token de acceso personal . Asegúrate de que tenga el permiso `repo`.
  3. Guardar Configuración: Completa el formulario de la izquierda con tu usuario, el nombre del repositorio y el token que generaste.
